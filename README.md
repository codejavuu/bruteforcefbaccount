
# Darkfb
Bruteforce facebook target account


## Installation



```bash
  git clone github.com/codejavuu/bruteforcefbaccount
  cd Bruteforcefbaccount
  python3 darkfb.py
```
    
## Usage/Examples

```
 put all your password in to passwords.txt
 install requirements.txt

```


## Features

- without token,id
- fast and easy to use
